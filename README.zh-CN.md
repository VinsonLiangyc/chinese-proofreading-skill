# 中文校对 Skill · 适用于 Claude

一个帮你在发布前抓出错别字、标点问题和前后不一致的 Claude Skill。

专为没有专职编辑的中文创作者和品牌团队而做。

---

## 问题是什么

稿子写完了，你自己读了三遍，还是漏掉了第二段那个错别字。

这不是粗心——这是大脑的工作方式。我们会自动脑补熟悉的文字。专业编辑存在的意义就在于此：作者没办法可靠地校对自己写的东西。

大多数中文创作者和品牌团队，没有这个人。

---

## 这个 Skill 做什么

把任何中文文本发给它，它运行六项检查，返回四段输出：

**① 修订后文本** — 完整输出修改后版本，结构不变，文风不改

**② 修改清单** — 每一处改动逐条列出，没有隐藏修改

**③ 修改原因** — 每处改动对应的具体规范依据（标点国标、数字用法、高频别字表等）

**④ 体例一致性检查** — 扫描全文是否存在同一事物写法不统一的情况：品牌名称、中英术语混用、数字格式、引号风格、人物称谓

---

## 适合谁用

**自媒体创作者** — 公众号作者、小红书创作者、B站专栏、播客文字稿编辑。一个人做内容、没有编辑把关时，这是你发布前的最后一道质检。

**品牌与公关团队** — 新闻稿、产品发布文案、品牌声明、社交媒体内容。官方渠道出现一个错别字，就会变成截图在网上流传。多人协作的文件，体例一致性检查尤其有用。

---

## 检查哪六个维度

| 维度 | 标准依据 |
|------|---------|
| 标点符号 | GB/T 15834—2011 |
| 数字写法 | GB/T 15835—2011 |
| 高频别字 | 《咬文嚼字》100个最常见别字（松弛感不是松驰感，震撼不是震憾，宣泄不是渲泄……） |
| 混淆词辨析 | 20+组：营利/盈利、定金/订金、作/做、即/既…… |
| 中英夹用 | CY/T 154—2017 |
| 体例一致性 | 专有名词、术语中英混用、数字格式、标点风格、称谓人称 |

这个 Skill 不改写，只纠错、只标注不一致。你的文风还是你的。

---

## 如何触发

说以下任意一句话即可：

> 「帮我校对」·「校对一下」·「有没有错别字」·「发布前检查一下」·「对外文案校对」·「新闻稿校对」·「帮我检查体例」·「前后用词一致吗」

Skill 不会自动触发，只在你明确要求时运行。

## 安装方式

### Claude.ai 项目

1. 打开 [claude.ai](https://claude.ai)，进入你的 Project
2. 点击 **Project Settings** → **Skills** → **Upload Skill**
3. 上传 `zh/SKILL.md`
4. 完成——说「帮我校对」即可触发
