# 中文文本校对 Skill

## 触发条件

仅在用户**明确说出**以下类似指令时启动：

**个人创作场景**
- 「帮我校对」「校对一下」「做校对」
- 「检查语法」「语法有没有问题」
- 「标点对不对」「检查标点」「有没有错别字」

**商业/品牌场景**
- 「发布前检查一下」「对外文案校对」「品牌文案校对」
- 「新闻稿校对」「公告校对」「声明校对」
- 「帮我检查体例」「前后用词一致吗」「统一一下格式」

用户未说出上述指令时，**不得主动启动**本 skill，即使文本中存在可改之处。

## 校对原则

- **保留原意**，不随意改写风格或润色表达
- **优先纠正**：错别字、病句、搭配不当、成分残缺、指代不清、标点错误、数字格式错误
- **不过度润色**：若原文语意清晰、无语法问题，不改动
- 用户的个人风格词汇（口语化表达、特定栏目用语）视为有意为之，不纳入修改
- 中英夹用文本以中文编辑规范为主，英文内容内部遵循英文规范
- **每次校对均须执行体例一致性检查**，无论用户是否单独提出

---

## 输出格式（四段式，缺一不可）

### A. 修订后文本
输出完整的校对后文本，保持原文段落结构。

### B. 修改清单

| # | 原文 | 修改后 |
|---|------|--------|
| 1 | … | … |

若无需修改，注明「本文未发现需修改之处」。

### C. 修改原因

| # | 修改原因 |
|---|----------|
| 1 | 【类别】具体说明 |

类别标签：【词典】【语法】【异形词】【标点】【数字】【中英夹用】【混淆词】

### D. 体例一致性检查

每次校对结束后，**必须**执行此步骤，检查全文是否存在同一事物写法前后不统一的情况。

检查维度包括：

**1. 专有名词与品牌名称**
- 人名、公司名、产品名是否全文统一（如：「苹果公司」与「Apple」混用）
- 简称是否在首次出现时定义，之后是否一致沿用（如：「香港中文大学」首次出现后简称「中大」）

**2. 中英文术语混用**
- 同一概念是否在不同段落交替使用中文与英文（如：「AI」与「人工智能」混用、「content」与「内容」混用）
- 建议：同一篇文章选定一种写法，全文统一

**3. 数字格式**
- 同类数字是否统一使用汉字或阿拉伯数字（如：同一篇文章「三个月」与「3个月」混用）

**4. 标点风格**
- 引号是否全文统一（「」与""不可混用）
- 破折号写法是否统一（「——」与「-」不可混用）

**5. 称谓与人称**
- 对同一人物的称呼是否前后一致（如：「先生」与全名交替使用是否有规律）
- 文章中的第一人称视角是否统一

**输出格式：**

若发现不一致：

| # | 不一致项目 | 出现位置 | 建议统一为 |
|---|-----------|---------|-----------|
| 1 | … | … | … |

若无不一致，注明「本文体例一致，无需调整」。

## 校对要点

### 一、标点符号（GB/T 15834—2011）

**基本形式**
- 所有标点须使用全角字符
- 句号用「。」，不用「.」
- 省略号用「……」（六点全角），不用「…」（三点）
- 破折号用「——」（两个全角连字符），不用「—」或「-」
- 标点前后无须留空格

**引号与书名号**
- 中文引号：「」（单层）；引文内部再有引文用『』（双层）
- 讲座、展览、栏目名用引号「」
- 书名、影视、专辑、报刊名用书名号《》，不用引号代替
- 书或杂志内的文章名用篇名号〈〉
- 英文书名、报刊名：以斜体标示，不借用《》
- 英文文章标题：用中文引号「」标示，内部保留正体

**顿号、逗号、分号**
- 并列成分之间用顿号「、」；分句之间用逗号「，」
- 并列成分已带引号时，顿号可省略
- 复句中分句关系较复杂时，用分号「；」隔开各分句

**冒号**
- 冒号「：」后所管范围延伸至该句末尾，不跨段使用
- 中英夹用时，英文作为中文的说明，中文后用冒号「：」；英文内部有冒号保留英文「:」

---

### 二、数字写法（GB/T 15835—2011）

**用汉字的情况**
- 叙述性、非精确数字（如：几十年、两三个）
- 固定词组、成语、惯用语中的数字（如：五四运动、三分之一、十万火急）
- 概数、约数相邻两数字并列时（如：三四个月、七八十年代），中间不加顿号
- 用「多」「余」「左右」「约」表示的约数（如：四十多人、五千余里）
- 世纪、星期（如：十九世纪、星期五）
- 农历、传统年号（如：农历八月十五、乾隆十二年）
- 年份用汉字时，「〇」不可写作阿拉伯数字「0」（如：二〇二四年，不是二0二四年）

**用阿拉伯数字的情况**
- 精确统计数字、编号、排名、评分（如：豆瓣6分、第3集）
- 日期、时间（如：2024年7月8日、晚上10时30分）
- 年度范围：用「2005至06年度」，不用「2005至2006年度」

**百分比**
- 一般叙述中宜写成中文（如：百分之三十）
- 统计比较类文章可用阿拉伯数字（如：30%）；全文须统一

### 三、常见别字（据《咬文嚼字》高频别字整理）

以下为当代出版物中差错率最高的别字，校对时须重点核查：

| 错误写法 | 正确写法 | 说明 |
|----------|----------|------|
| 松驰感 | 松弛感 | 「弛」指放松弓弦，「驰」指疾行 |
| 震憾 | 震撼 | 「撼」指摇动，「憾」指遗憾 |
| 渲泄 | 宣泄 | 「宣」指疏导，「渲」指渲染 |
| 穿流不息 | 川流不息 | 「川」指河流，不是「穿」 |
| 渡假村 | 度假村 | 「度」指过（时间），「渡」指渡河 |
| 一幅对联 | 一副对联 | 量词：「副」用于成对或成套的东西 |
| 既使 | 即使 | 「即使」表假设，「既然」表已然 |
| 挖墙角 | 挖墙脚 | 固定词组 |
| 再接再励 | 再接再厉 | 「厉」指磨砺 |
| 谈笑风声 | 谈笑风生 | 固定成语 |
| 食不裹腹 | 食不果腹 | 「果」出自《庄子》，指吃饱 |
| 过度（政府） | 过渡（政府） | 「渡」指由一阶段转入另一阶段 |
| 作为（误用） | 做为 × → 作为 ✓ | 「作为」是正确词形，「做为」为误写 |
| 启示（招聘） | 启事 | 「启事」指公告文体；「启示」指启发 |
| 营利/盈利 | 二者有别 | 「营利」指谋取利润（动词）；「盈利」指获得盈余（名词） |
| 订金 | 定金 | 「定金」有法律效力；「订金」无 |
| 即/既 | 二者有别 | 「即」有未然义；「既」有已然义 |
| 黏/粘 | 二者有别 | 「黏」（形容词，有黏性）；「粘」（动词，粘贴） |
| 蜇人 | 蛰人 × → 蜇人 ✓ | 蜜蜂「蜇」人；动物「蛰伏」 |
| 泄密 | 泄密 ✓（「泄秘」为误写） | — |

### 四、常见混淆词辨析

**形近义近，易用错**

| 词组 | 辨析要点 |
|------|----------|
| 年轻 / 年青 | 比较年龄用「年轻」；表青春朝气用「年青」 |
| 考查 / 考察 | 「考查」用标准评定；「考察」指实地观察研究 |
| 国是 / 国事 | 「国是」指国家大计，书面庄重语；「国事」口书面均可用 |
| 公信力 / 公信度 | 「公信力」论大小（增强/减弱）；「公信度」论高低（提高/降低） |
| 陈规 / 成规 | 「陈规」含贬义；「成规」中性 |
| 狙击 / 阻击 | 「狙击」指隐蔽偷袭；「阻击」指阻拦敌方进攻 |
| 无时无刻 | 须与「不」连用，表示「每时每刻」；不可单独表示「每时每刻」 |
| 家父 / 令尊 | 「家父」谦称自己父亲；「令尊」敬称他人父亲 |

**「作」与「做」**
- 「作」多用于抽象、书面（作品、作者、作废、作为、操作、作孽）
- 「做」多用于具体动作（做饭、做事、做生意、做梦）
- 「作为」是固定词形，「做为」为误写

---

### 五、词语搭配与病句

**常见搭配问题**
- 「背景」后接处所或情境，宜用「设定在」，而非「背景是」
- 「观感」「感受」可搭配「整体」「总体」
- 「向来」表一贯如此，后接肯定或否定均可，注意语义连贯
- 「不禁」后接动词，不用「不禁地」
- 「戴上紧箍咒」→「念紧箍咒」（咒语是念的，不是戴的）

**「的」「地」「得」三字**
- 修饰名词用「的」（美丽的风景）
- 修饰动词用「地」（认真地写）
- 补充说明程度结果用「得」（写得很好）

**虚词误用**
- 「由于」引导原因；「对于」引导对象，不混用
- 关联词须成对出现（「不仅……还……」「虽然……但……」）
- 「无时无刻」须与「不」连用

**成分残缺**
- 长句中注意主语是否缺失或偷换
- 注意谓语或宾语是否完整

### 七、中英夹用规范（CY/T 154—2017）

**总则**
- 以中文编辑规范为主，英文内容内部遵循英文规范

**标点选用**
- 中文句中夹用英文单词或词组，整句以中文标点结尾
- 夹用完整英文句子时，用中文引号「」标示，英文内部保留英文标点，整句以中文句号结尾
- 中英文之间需破折号时，用中文「——」；英文内部用英文「－」
- 中文省略号「……」（六点）；英文省略号「...」（三点），不混用

**字母大小写**
- 中文句中夹用普通英文单词，无论位于何处，首字母一律小写
- 专有名词（人名、地名、品牌名）保留首字母大写
- 夹用完整英文句子时，首单词首字母保留大写

**空格**
- 中英文之间可根据视觉效果决定是否留空格，但全文须保持统一
- 英文词句紧接中文标点时，二者之间不留空格

**英文书刊名**
- 英文书名、报刊名在中文句中，用斜体标示，不借用《》
- 英文文章标题用中文引号「」标示，内部用正体
